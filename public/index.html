Welcome to the SaaS Pricing Scraper API. This service extracts pricing plans from SaaS websites.

Available Endpoints:

GET /api/hello

Simple health check endpoint

Response: {"message": "Hello from Vercel!", "status": "success"}

GET/POST /api/test-scrape

Returns mock pricing data for testing (no authentication required)

Response: JSON array of pricing plans

POST /api/scrape

Scrapes actual pricing data from a website

Headers: x-api-key: your-api-key

Body: {"url": "https://example.com/pricing"}

Response: JSON array with extracted pricing plans

Example Usage:

curl -X POST https://your-deployment.vercel.app/api/scrape \
  -H "Content-Type: application/json" \
  -H "x-api-key: test-key" \
  -d '{"url": "https://stripe.com/pricing"}'

Response Schema:

[
  {
    "plan_name": "Basic",
    "price": "$15/month", 
    "pricing_model": "Tiered",
    "features": ["Feature 1", "Feature 2"],
    "billing_cycle": "monthly"
  }
]
